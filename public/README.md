## 🛡️ What It Does

eBay Buyer Guardian is a Chrome extension that shows **color-coded risk badges** next to seller names on eBay:

- 🟢 **Green** — Seller looks safe (high feedback %, sufficient count, established account)
- 🟡 **Yellow** — Some risk factors detected (borderline feedback, low count, new account)
- 🔴 **Red** — High risk, proceed with caution (low feedback, very few reviews, brand new account)

## 🚀 Installation

1. **Download** the [ebay-buyer-guardian.zip](assets/ebay-buyer-guardian.zip)
2. **Unzip** the file to a folder on your computer
3. Open Chrome → `chrome://extensions` → Enable **Developer Mode**
4. Click **"Load unpacked"** → Select the unzipped folder
5. Visit any eBay listing — badges appear automatically!
